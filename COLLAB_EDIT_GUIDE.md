# 📋 在线协作编辑指南

本文档说明如何使用腾讯文档协作编辑噜咪扩展数据。

---

## 🚀 快速开始

### 1. 获取模板

下载 `extra-data-template.csv` 模板文件，或在腾讯文档中创建新表格。

### 2. 表格格式

| 噜咪ID | 体型 | 活动地图 | 关键特质 | 行为习惯 |
|--------|------|----------|----------|----------|
| 101001 | 小型 | 草原 | 善于奔跑，行动敏捷 | 白天活动，喜群居 |
| 101002 | 中型 | 森林 | 擅长攀爬，灵活 | 夜行性，独居 |

**字段说明：**
- **噜咪ID**：必填，噜咪的唯一标识（如 101001）
- **体型**：自由填写（如：小型、中型、大型、巨型）
- **活动地图**：自由填写（如：草原、森林、湿地）
- **关键特质**：自由填写，描述该噜咪的特点
- **行为习惯**：自由填写，描述该噜咪的行为模式

### 3. 同步到 Wiki

1. 在腾讯文档中完成编辑
2. 导出为 CSV 文件
3. 将文件重命名为 `extra-data.csv`
4. 放在项目根目录（与 `sync-extra-data.cjs` 同级）
5. 运行同步命令：

```bash
node sync-extra-data.cjs
```

### 4. 验证结果

同步成功后，访问 LumiWiki 图鉴页面，查看新增字段是否显示。

## 📝 编辑规范

### 噜咪ID 获取

- 在图鉴页面点击任意噜咪，URL 中的数字即为 ID
- 例如：`http://10.27.17.179:3005/#/lumi/101001` 中的 `101001`

> **🚨 重要：访问地址固定为端口 3005，严禁使用其他端口！**

### 文本填写建议

- 保持简洁明了，一句话描述
- 多个特点可用逗号、顿号分隔
- 避免使用特殊符号和换行

### 示例数据

| 噜咪ID | 体型 | 活动地图 | 关键特质 | 行为习惯 |
|--------|------|----------|----------|----------|
| 101001 | 小型 | 草原 | 善于奔跑，行动敏捷 | 白天活动，喜群居 |
| 101101 | 大型 | 湿地 | 力量强大，防御力高 | 喜欢在水边休息 |
| 108501 | 超大型 | 山地 | 攻击力极强，性格暴躁 | 独居，领地意识强 |

---

## ⚠️ 注意事项

1. **不要修改表头**：保持第一行为 `噜咪ID,体型,活动地图,关键特质,行为习惯`
2. **ID 必须正确**：错误的 ID 会导致数据无法关联
3. **定期备份**：重要更新前建议备份 `extra.json` 文件
4. **同步权限**：需要有项目目录的写入权限

## 🔧 故障排查

### 问题：同步脚本报错 "未找到 extra-data.csv"

**解决方案**：
- 检查文件名是否正确
- 确认文件在项目根目录

### 问题：数据同步后页面不显示

**解决方案**：
- 硬刷新浏览器（Ctrl+F5）
- 检查浏览器控制台是否有错误

### 问题：部分数据没有同步

**解决方案**：
- 检查 CSV 中对应行的 ID 是否正确
- 确认 ID 列没有多余空格
