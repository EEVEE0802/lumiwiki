### 在线协作编辑方案（原计划）

**状态**：待与同事对齐确认

#### 方案概述

使用在线协作表格（腾讯文档/飞书表格）维护游戏数据，通过定时脚本同步到 JSON 文件，网站读取 JSON 展示。

#### 架构图

```
┌─────────────────┐     ┌──────────────┐     ┌─────────────┐
│  腾讯文档/飞书   │ --> │  同步脚本     │ --> │  JSON 文件   │
│     表格        │     │  (定时执行)   │     │  (数据源)    │
└─────────────────┘     └──────────────┘     └─────────────┘
                                                       │
                                                       ▼
                                              ┌─────────────┐
                                              │  LumiWiki   │
                                              │   网站       │
                                              └─────────────┘
```

#### 优点

- ✅ 同事熟悉操作，无需学习成本
- ✅ 支持实时协作，多人可同时编辑
- ✅ 有版本历史，可回溯
- ✅ 开发量最小，可快速落地
- ✅ 无需维护复杂的后端服务

#### 缺点

- ⚠️ 数据结构受表格格式限制
- ⚠️ 需要编写数据转换逻辑
- ⚠️ 依赖第三方服务（内网可用企业微信/飞书）
- ⚠️ 定时同步存在延迟（非实时更新）
