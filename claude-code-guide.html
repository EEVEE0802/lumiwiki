🧠 第一部分：AI 基础概念

🤖

Agent（智能体）

能够感知环境、做出决策并执行行动的 AI 系统，具有自主性和目标导向性

🛠️

Skill（技能）

AI Agent 具备的特定能力，如代码生成、文件操作、数据分析等可复用的功能模块

🧠

大模型（LLM）

大规模语言模型，通过海量数据训练，具备理解和生成人类语言的能力

🔑

API Key（密钥）

访问 AI 服务的身份验证凭证，相当于你的"会员卡"，用于计费和权限控制

📊

Token（词元）

AI 模型处理文本的基本单位，可以是一个字、词或词的一部分，是计费的基础

🎯

Prompt（提示词）

用户向 AI 发送的指令或问题，用于引导 AI 生成期望的响应

🔗 概念之间的关系

AI 生态系统的运作流程
💬 你的 Prompt
→
🔑 API Key 验证
→
🤖 Agent 接收任务
→
🧠 调用大模型
→
🛠️ 执行 Skill 操作
→
✅ 返回结果
💡 计费说明：Token 数量 = 输入 Token + 输出 Token
💡 核心关系：Agent 是"大脑"，使用大模型思考，用 Skill 执行具体任务（读文件、写代码、运行命令等）

🚀 第二部分：Claude Code 安装与使用

📦 前置准备：安装 Node.js

Claude Code 需要 Node.js 环境，请先下载安装：

# Node.js 官方下载地址（选择 LTS 版本）
https://nodejs.org/

# 或直接下载 Windows 安装包
https://nodejs.org/dist/v20.18.2/node-v20.18.2-x64.msi
💡

验证安装：在终端输入 node --version，显示版本号即安装成功

🚀 安装 Claude Code

# 使用 npm 全局安装
npm install -g @anthropic-ai/claude-code
💡

系统要求：Windows 10+ / Node.js 18+

💻 Claude Code 基本使用

🚀 三步启动 Claude Code

  • 打开 Windows Terminal — 按下 Win + R，输入 wt 回车
  • 进入项目目录 — 使用 cd 命令切换到你的项目文件夹
  • 输入 claude 启动 — 直接输入 claude 并回车

📝 常用命令

  • /help — 查看帮助信息
  • /clear — 清空对话历史
  • /exit 或 Ctrl+C — 退出程序
  • /attach — 附加文件到上下文
💡

首次使用：第一次启动会提示配置 API Key，按提示输入即可，之后就可以愉快地对话啦！

🔑 第三部分：API Key 获取与配置

📝 获取 API Key

  • 访问 Anthropic 官网：console.anthropic.com
  • 注册/登录账号
  • 进入 "API Keys" 页面
  • 点击 "Create Key" 创建新的 API Key
  • 重要：立即复制保存，Key 只显示一次！
⚠️

安全提醒：不要将 API Key 提交到公开仓库或分享给他人！

⚙️ 配置 API Key

方法一：环境变量（推荐）

# Windows PowerShell
$env:ANTHROPIC_API_KEY = "your-api-key-here"

# Windows CMD
set ANTHROPIC_API_KEY=your-api-key-here

# macOS / Linux
export ANTHROPIC_API_KEY="your-api-key-here"

方法二：配置文件永久保存

# 创建配置文件
claude config set api_key your-api-key-here

💡 实用技巧与最佳实践

📌 高效使用技巧

  • 明确指令：清晰描述你的需求，避免模糊表述
  • 提供上下文：使用 /attach 附加相关文件
  • 分步执行：复杂任务可以分解为多个小步骤
  • 验证结果：AI 生成的代码需要测试和验证
  • 保存记录：重要的对话可以保存到文档中

🎯 最佳实践

  • 在项目根目录创建 CLAUDE.md 说明项目结构
  • 使用 .gitignore 排除敏感文件和临时文件
  • 定期更新 API Key 使用配额监控
  • 结合 Git 使用，让 AI 帮助你进行代码审查
